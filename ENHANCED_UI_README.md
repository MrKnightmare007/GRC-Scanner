## 🛠️ Installation & Setup

### 1. Install Dependencies
```bash
cd GrcScanner/frontend
npm install
```

### 2. Start the Application
```bash
# Terminal 1 - Backend
cd GrcScanner/backend
python app.py

# Terminal 2 - Frontend
cd GrcScanner/frontend
npm start
```

### 3. Access the Application
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000
